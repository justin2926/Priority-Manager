## Priority Manager
CS 122 Task Management System Project

# Description
A task management system built in python that helps users organize their tasks with a priority level feature. App allows users to create, modify, and delete tasks.

# Features
- Add, modify, delete tasks
- Specifiy priority of task
- Set status (pending/complete) of task
- Sort by status and priority

# Tools used
- TKinter: GUI
- SQLAlchemy/SQLite3: Database
- Python: App logic

# Prerequisites
- Python3 (Latest)
